# Spotify Data Analysis 

## Project Overview
This project focuses on cleaning, processing, analyzing, and visualizing Spotify track data using Python.
The dataset contains information about Spotify songs including:
- track names
- artists
- genres
- popularity
- danceability
- energy
- tempo
- valence
- loudness
- and other audio features.
The goal of this project is to perform Data Cleaning and Exploratory Data Analysis (EDA) to generate meaningful insights from Spotify music data.

## Objectives

- Perform data cleaning
- Handle missing values
- Remove duplicate records
- Detect outliers
- Analyze Spotify audio features
- Create visualizations using Python
- Generate insights through storytelling with data

## Tools & Libraries Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning Performed

The following preprocessing steps were completed:
- Removed unnecessary columns
- Checked missing values
- Removed duplicate rows
- Performed outlier analysis
- Verified dataset structure and data types

## Visualizations Included

1. Top Genres Distribution
2. Song Popularity Distribution

5. Correlation Heatmap
6. Tempo Distribution
7. Top 10 Artists
8. Valence vs Energy
9. Song Duration Distribution
10. Popularity by Genre
11. Outlier Detection using Boxplot

## Key Insights

- Certain genres dominate Spotify listening trends.
- Most songs fall within medium popularity ranges.
- Danceability and energy show positive correlation.
- Non-explicit songs form the majority of tracks.
- Most songs have durations between 2–5 minutes.
- Energy and loudness are strongly related audio features.
